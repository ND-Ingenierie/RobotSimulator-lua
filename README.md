# Robot Simulator - Public Resources

This repository contains public examples, scripts, and levels for the Robot Simulator 2D application.

## Contents

- **examples/scripts/** - Example Lua scripts demonstrating robot programming
- **examples/levels/** - Sample level files for testing and learning

## Getting Started

1. Download the Robot Simulator 2D binaries from our official website
2. Clone this repository to access example content
3. Load scripts and levels directly from the `examples/` directory

## Contributing Examples

We welcome community contributions! If you've created interesting levels or scripts:
1. Fork this repository
2. Add your content to the appropriate `examples/` subdirectory
3. Submit a pull request with a description

## Reporting Issues

Please use the [Issues](../../issues) section of this repository to report bugs, request features, or ask questions about the Robot Simulator.
