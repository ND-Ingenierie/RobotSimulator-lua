# Example Levels

This directory contains example `.rs_level` files that you can load in the Robot Simulator.

## Creating Your Own Levels

Use the Level Editor application to create custom levels:

1. Launch the Level Editor
2. Use drawing tools to create tracks and obstacles
3. Set the robot start position
4. Save as `.rs_level` file

## Level File Format

`.rs_level` files are binary files containing:
- Ground layer image data (for line-following tracks)
- Obstacle layer image data (walls and barriers)
- Robot start position and rotation

## Contributing

Feel free to contribute your own interesting level designs! Submit a pull request with:
- The `.rs_level` file
- A brief description of the level's purpose or challenge
- Optional: A screenshot or description of the intended solution
